## Running Tests Locally

We use **pytest** for our unit and integration testing. Before submitting a Pull Request, please ensure all tests pass on your local machine.

### Prerequisites

1.  Ensure you have a Python environment set up.
2.  Install all project dependencies, which include the testing tools:
    ```
    pip install -r requirements.txt
    ```

### Execution

To run the entire test suite, simply execute the following command from the project root directory:

```
pytest

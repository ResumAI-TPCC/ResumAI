# ResumAI Frontend

Match Analysis page - React frontend with Tailwind CSS.

## Tech Stack

- React 18
- Vite
- Tailwind CSS

## Getting Started

```bash
# Install dependencies from lockfile
npm ci

# Start development server
npm run dev

# Build for production
npm run build

# Add a new package
npm install <package-name>

# Verify linting before commit
npm run lint
```
